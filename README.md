# Excel Redirect Test

Simple test to identify the redirect limit for any browser, meant to be used
specifically from within Microsoft Excel's web queries.

## Installation

- Clojure
- Leiningen

## Usage

Compile and run:

'''lein run'''

Then point to:

'''localhost:4895'''
